1. Introduction

The IoT Agriculture Rover is a smart farming assistant designed to monitor soil conditions, collect data, and assist in automating field operations. This manual will guide you through setting up, using, and maintaining your rover.

3. Using the Rover

After setup, use the web or mobile app to control the rover. Features include:

  • Manual or autonomous navigation
  • Live soil data monitoring
  • Camera feed (if equipped)
  • Data logging to cloud

4. Dashboard Guide

The dashboard allows real-time monitoring and control. You can view:

  • Sensor values (moisture, temperature, etc.)
  • GPS location of the rover
  • Navigation controls
  • Data history and analytics

5. Maintenance

  • Keep sensors clean and dry.
  • Recharge batteries after each use.
  • Store in a dry place when not in use.

6. Troubleshooting

  • Rover not moving: Check battery and motor connections.
  • No sensor data: Restart the system and check wiring.
  • Connection lost: Ensure device is within range and Wi-Fi is stable.
