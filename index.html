Overview

This Agriculture Rover is a working prototype that demonstrates the core functionalities of autonomous farming robots. Controlled via Bluetooth using a custom Android app, the rover can navigate through fields, pump water, and seed soil with simple and intuitive commands.

The rover is powered by Arduino Uno and built with 3D-printed PLA components, showcasing the potential of rapid prototyping in agriculture tech.

Key Features

  • Bluetooth-based control via custom Android application.
  • Pumps water using a mini 5V pump on command.
  • Performs seeding through a servo-controlled seed valve (SG90).
  • Lightweight, compact design using 3D-printed PLA parts.
  • Powered by two 18650 cells; rechargeable via 12V input (max 5 mins).

Hardware Components

  • Arduino Uno
  • HC-05 Bluetooth Module
  • SG90 Servo Motor (for seed valve)
  • Mini 5V Water Pump (for irrigation)
  • Relay Module

  • 3D Printed Parts: Water Tank, Arduino Case, Seed Container (PLA)

// Bluetooth: RX (D10), TX (D11)
SoftwareSerial BT(10, 11); // RX, TX

// Motor pins
const int IN1 = 2;
const int IN2 = 3;
const int IN3 = 4;
const int IN4 = 5;

// Relay pin
const int relayPin = 8;

// Servo pin
const int servoPin = 9;

Servo seedValve;

void setup() {
  // Motor setup
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);

  // Relay setup
  pinMode(relayPin, OUTPUT);
  digitalWrite(relayPin, HIGH); // Pump OFF

  // Servo setup
  seedValve.attach(servoPin);
  seedValve.write(0); // Closed position

  // Serial monitor
  Serial.begin(9600);
  Serial.println("Serial monitor ready.");

  // Bluetooth
  BT.begin(9600);
  Serial.println("Bluetooth ready on SoftwareSerial. Awaiting commands...");
}

void loop() {
  if (BT.available()) {
    char cmd = BT.read();

    // Log command
    Serial.print("Received Bluetooth command: ");
    Serial.println(cmd);

    switch (cmd) {
      case 'F': moveForward(); Serial.println("Action: Move Forward"); break;
      case 'B': moveBackward(); Serial.println("Action: Move Backward"); break;
      case 'L': turnLeft(); Serial.println("Action: Turn Left"); break;
      case 'R': turnRight(); Serial.println("Action: Turn Right"); break;
      case 'S': stopMotors(); Serial.println("Action: Stop"); break;

      case 'O': seedValve.write(120); Serial.println("Action: Open Seed Valve (40Â°)"); break;
      case 'C': seedValve.write(0);  Serial.println("Action: Close Seed Valve (0Â°)"); break;

      case 'W': digitalWrite(relayPin, HIGH); Serial.println("Action: Water Pump ON"); break;
      case 'w': digitalWrite(relayPin, LOW);  Serial.println("Action: Water Pump OFF"); break;

      default: Serial.println("Warning: Unknown command"); break;
    }
  }
}

// Movement functions
void moveForward() {
  stopMotors();
  delay(200);
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, HIGH);
  digitalWrite(IN4, LOW);
}

void moveBackward() {
  stopMotors();
  delay(200);
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  digitalWrite(IN3, LOW);
  digitalWrite(IN4, HIGH);
}

void turnLeft() {
  stopMotors();
  delay(200);
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, HIGH);
  digitalWrite(IN3, HIGH);
  digitalWrite(IN4, LOW);
}

void turnRight() {
  stopMotors();
  delay(200);
  digitalWrite(IN1, HIGH);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW);
  digitalWrite(IN4, HIGH);
}

void stopMotors() {
  digitalWrite(IN1, LOW);
  digitalWrite(IN2, LOW);
  digitalWrite(IN3, LOW);
  digitalWrite(IN4, LOW);
}

Setup Instructions

  1. Charge the rover using a 12V power source for no more than 5 minutes.
  2. Download and install the custom Android app from the link above.
  3. Power on the rover using the main switch.
  4. Enable Bluetooth on your Android device.
  5. Scan for available devices and connect to HC-05 (may appear as "HC05").
  6. Open the app and select the connected HC-05 device from the list.
  7. Once connected, you can control navigation, pump water, and trigger seeding using the appâ€™s intuitive controls.
Note: This is a prototype for demonstration purposes only. For real field deployment, further ruggedization and automation would be required.
