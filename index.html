1. Introduction

The IoT Agriculture Rover is a smart field robot designed to monitor soil conditions, assist with data-driven farming, and provide automated support in agricultural operations. This manual walks you through setup, usage, maintenance, and troubleshooting.

2. Getting Started & Setup

  • Included: Rover unit, battery pack, charger, soil sensors, camera module (optional), user guide.
  • Requirements: Smartphone/PC with internet, mobile app or web dashboard access.
  • Assembly: Attach sensors, insert battery, power on using the main switch.
  • App Setup: Install the mobile app or go to the web dashboard. Follow on-screen pairing instructions.

3. Using the Rover

After successful setup, use the app to control and monitor the rover:

  • Switch between manual and auto modes.
  • View real-time sensor data (moisture, temperature, humidity).
  • Track location using GPS (if enabled).
  • Record data logs and sync to the cloud.

4. Dashboard Guide

  • Sensor Display: Real-time readings.
  • Control Panel: Joystick or route selection.
  • Map View: Live GPS position of rover.
  • Analytics: Charts for historical data.

5. Maintenance

  • Clean sensors after each use.
  • Keep electronics dry.
  • Recharge battery before storage.
  • Run diagnostics monthly.

6. Troubleshooting

  • No Power: Check battery connection or charge level.
  • No Sensor Data: Recheck sensor ports and reboot system.
  • Connectivity Issue: Verify Wi-Fi range and signal strength.
  • GPS Not Working: Ensure rover is in open outdoor space.
